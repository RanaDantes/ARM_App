About This Project

Apriori Insights is a web-based application developed as part of the Performance Innovation Task (PIT) for the EL103 Business Intelligence course. This project showcases the practical use of data mining techniques—particularly the Apriori algorithm—to uncover valuable patterns in transaction data through association rule mining.

The goal of this tool is to provide a lightweight and intuitive platform that allows users to upload transactional datasets, set mining parameters (such as minimum support and confidence thresholds), and receive immediate insights into purchasing behavior. It highlights the role of Business Intelligence (BI) in translating raw data into meaningful, actionable decisions—particularly in retail and e-commerce domains.

Beyond its academic purpose, Apriori Insights reflects a deeper understanding of how modern BI tools empower organizations to make data-informed strategies, streamline operations, and ultimately, drive performance through innovation.
